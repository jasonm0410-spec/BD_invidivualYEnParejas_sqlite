# BD_invidivualYEnParejas_sqlite

# Mis Proyectos de Bases de Datos Relacionales

¡Hola! En este repositorio encontrarás los scripts y archivos `.db` de dos bases de datos que desarrollé como parte de mis prácticas escolares. Estos proyectos me ayudaron a aprender cómo crear tablas, relacionarlas entre sí y gestionar información utilizando SQLite.

Las dos bases de datos están organizadas correctamente e incluyen **llaves primarias**, **llaves foráneas** y **eliminación en cascada**, lo que permite mantener la información relacionada de forma ordenada y evitar registros sin conexión.

---

## 1. Sistema de Gestión de Alumnos

Este fue uno de los primeros proyectos que realicé. Su objetivo es representar el control de estudiantes, materias y profesores dentro de un centro educativo. La estructura está diseñada para evitar la duplicación de información y facilitar la administración de los datos.

## 2. Sistema de Biblioteca

Este proyecto simula el funcionamiento de una biblioteca donde los usuarios pueden solicitar préstamos de libros. Además, incorporé una tabla de **categorías** para organizar los libros según su género o tipo, haciendo la base de datos más completa y fácil de consultar.

---

### 💻 ¿Cómo utilizar estos archivos?

Los archivos `.db` pueden abrirse directamente en páginas como `sqliteonline.com` para visualizar las tablas y realizar consultas SQL. Si deseas revisar el código, también puedes abrir los archivos `.sql` con cualquier editor de texto para ver las instrucciones `CREATE TABLE`, `INSERT` y demás sentencias utilizadas en la creación de las bases de datos.
